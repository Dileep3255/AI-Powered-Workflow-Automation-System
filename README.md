# Manufacturing Workflow Automation

OCR-based document text extraction system using React, Flask, and Tesseract OCR.

## Features

- Upload image documents
- Extract text using OCR
- Detect Date and Quantity
- Structured output display

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Flask
- pytesseract
- Pillow

## Run Frontend

```bash
npm install
npm run dev

```

## Run Backend

```bash
pip install flask flask-cors pillow pytesseract
python app.py
```
